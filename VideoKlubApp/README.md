# Video Klub App

Windows Forms aplikacija za upravljanje video klubom u C#.

## Funkcionalnosti
- Upravljanje klijentima (dodavanje, brisanje)
- Upravljanje filmovima (dodavanje, brisanje)
- Zaduživanje i vraćanje filmova
- Pregled svih zaduženja

## Tehnologije
- C# WinForms
- SQL Server (LocalDB)
- ADO.NET

## Baza podataka
Baza se zove `VideotekaDB` sa tabelama:
- `Klijenti`
- `Filmovi`
- `Zaduzenja`

## Kako pokrenuti
1. Pokreni SQL skriptu u SSMS
2. Otvori VideoKlubApp.sln u Visual Studio
3. Pokreni aplikaciju (F5)
